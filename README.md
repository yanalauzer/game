# 2048

веб-версия игры 2048, где пользователь имеет возможность:

Взаимодействовать с игровым полем  
Записать свой рекорд в таблицу лидеров в случае окончания игры  
Просмотреть список лидеров  
Продолжить игровую сессию с последнего действия  
Вернуться на один шаг назад  
Начать игру заново  

## Был реализован следующий функционал:

### Макет страницы
-Создатна базовая HTML-страница игры  
-Создан контейнер, который будет показываться при окончании игры  
-При звершении игры отображается следующая информация - информационное сообщение, инпут для ввода имени, кнопка "сохранить результат", кнопка "начать заново"   
-Адативный дизайн и медиазапросы.   
-На десктопе управление стрелочками на клавиатуре  
-На мобильных версиях отрисованы 4 кнопки управления (вверх, вправо, вниз, влево).  


### Логика приложения
-Реализация движений вверх, влево, вправо, вниз  
[4, 4, 4, 4] -> [16, 0, 0, 0] (движение влево) (+32 очка - +8, +8, +16)
[16, 16, 2, 2] -> [32, 4, 0, 0] (движение влево) (+36 очков - +32, + 4)  
-Реализация проверки окончания игры (нет возможных движений)  
-Появление 1-2 новых плиток с значениями 2/4 на случайных пустых клетках  
-Отображение текущего счета пользователя  
-Реализация отмены хода   
-Сохранение состояния игры, сохранения лидерборда
